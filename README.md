## Project Setup

```sh

```

### Install Editor files

```sh
node node_modules/datatables.net-editor/install.js <path-to-editor-zip-files>
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
